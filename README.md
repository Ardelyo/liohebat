# Project Overview

This project consists of the following files:

## index.html
The main HTML file for the project.

## package-lock.json
The lock file for the project's dependencies.

## package.json
The package file for the project, containing metadata and dependencies.

## script.js
The JavaScript file for the project, containing the main logic.

## style.css
The CSS file for the project, containing the styles for the HTML elements.
